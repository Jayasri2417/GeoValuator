# GeoValuator

Hybrid microservices blueprint with React frontend, Node/Express core, and FastAPI AI engine.

## Frontend
- Location: `client/`
- Stack: React + Vite + Tailwind + React-Leaflet + Lucide icons.
- Scripts: `npm install`, `npm run dev` (default Vite port 5173).

## Backend (planned)
- Location: `server/` (placeholder for Express/Mongo service).

## AI Engine (planned)
- Location: `ai_engine/` (placeholder for FastAPI service).

Refer to the client folder for the implemented UI.
